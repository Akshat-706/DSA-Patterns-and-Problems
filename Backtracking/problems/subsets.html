Subsets (Power Set)

LeetCode [Medium]

Problem Statement

Given an integer array nums of unique elements, return all possible subsets (the power set). The solution set must not contain duplicate subsets. Return the solution in any order.

Example: Input: nums = [1,2,3] â†’ Output: [[],[1],[2],[1,2],[3],[1,3],[2,3],[1,2,3]]

Approach 1: Backtracking (Recursive)

Explanation (Brute/Standard): Classic "choose/don't choose" recursive backtracking. For each element, decide to include it or not, and recurse. When reaching the end, add the current subset.

Time Complexity: O(N * 2^N)

Space Complexity: O(N) recursion stack

function subsets(nums):
    result = []
    backtrack(result, [], nums, 0)
    return result

function backtrack(result, current, nums, start_index):
    add copy of current to result
    
    for i from start_index to length(nums)-1:
        current.push(nums[i]) // Include
        backtrack(result, current, nums, i + 1)
        current.pop() // Backtrack (Don't include)

Approach 2: Cascading (Iterative)

Explanation (Better): Start with an empty subset. For each number, add it to all existing subsets to generate new subsets, effectively doubling the total subsets each iteration.

Time Complexity: O(N * 2^N)

Space Complexity: O(N * 2^N) for output

function subsets(nums):
    result = [[]] // Start with empty subset
    for num in nums:
        new_subsets = []
        for current_subset in result:
            new_subset = copy(current_subset)
            new_subset.push(num)
            new_subsets.push(new_subset)
        result.extend(new_subsets)
    return result

ðŸ’¡ Think: Iteratively add each number to existing subsets to build all possible subsets.

Approach 3: Bit Manipulation

Explanation (Optimal): Map each subset to a binary number from 0 to 2^N - 1. Each bit represents whether the element at that index is included in the subset.

Time Complexity: O(N * 2^N)

Space Complexity: O(N * 2^N) for output

function subsets(nums):
    n = length(nums)
    result = []
    for i from 0 to (2^n - 1):
        subset = []
        for j from 0 to n-1:
            if (i >> j) & 1:
                subset.push(nums[j])
        result.push(subset)
    return result

ðŸ’¡ Think: Use bits of numbers 0..2^N-1 to represent subsets; 1-bit means include the element.
