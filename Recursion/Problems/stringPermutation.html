Approach 2: Using Used Array

Explanation: Build permutation step by step, marking characters used.

function backtrack(path, used):
    if len(path) == n:
        add path copy to result
        return
    for i in 0..n-1:
        if used[i]:
            continue
        used[i]=True
        backtrack(path + s[i], used)
        used[i]=False
